Program ini dibuat untuk membantu pengguna mencatat pengeluaran harian berdasarkan kategori seperti Makan, Transport, Hobi, dan Lainnya. Data pengeluaran dikelompokkan secara otomatis dan ditampilkan dalam bentuk ringkasan total per kategori dan total keseluruhan.

  Fitur Utama Program
- Menggunakan Bahasa Pascal
- Menggunakan Record untuk menyimpan struktur data pengeluaran  
- Menggunakan Array untuk menyimpan banyak data  
- Menggunakan Function & Procedure  
- Menggunakan Looping (For)  
- Menggunakan If else
